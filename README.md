# 🌐 Portfolio Web - API Rest (Update)

Este projeto é uma evolução do meu portfólio pessoal, focado no desenvolvimento do back-end. A aplicação foi transformada em uma API REST para disponibilizar os dados de forma limpa e organizada, separando a lógica de dados de qualquer futura integração com o front-end.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python
* **Framework Web:** Django
* **Desenvolvimento de API:** Django Rest Framework (DRF)
* **Banco de Dados:** SQLite (padrão de desenvolvimento)

## ⚙️ O que foi implementado

* **Endpoints REST:** Criação de rotas da API para servir os dados do portfólio de forma estruturada.
* **Serializers:** Conversão dos dados do banco de dados para o formato JSON.
